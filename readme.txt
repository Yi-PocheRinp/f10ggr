【はじめに】
　クリップボードにコピーしたテキストを、F10キーを押した際にググるソフトウェアです。
　起動するとタスクバーに常駐し、F10キーを押したときにクリップボードに保持されているテキストをGoogleで検索します。

【使い方】
  exeファイルをダブルクリックしてください。
　タスクトレイに常駐します。
　F10キーを押すとクリップボードのテキストをGoogleで検索します。
　ShiftキーとF10キーを同時に押すことで終了します。
　検索エンジンやホットキーの変更などの使い方については同梱している説明書.pdfをお読みください。

【注意事項】
 Plug-insフォルダは削除しないでください。動作不良の原因となります。
